Help

Network format

Network must be represented in edge-list format

1 3
2 3
3 5

Weighted

If edges are weighted, select Yes.

Self edges

If network contains self-edges (edge from node A to node A), select Yes.

Redundant edges

If network contains redundant-edges (edge from node A to node A), select Yes.

Queue limit

NS2 supports Queue Limit restriction. Choose accordingly.

Edge Direction

If edges are uni-directional, select Uni-directional.

Loss

If network contains self-edges (edge from node A to node A), select Yes.
